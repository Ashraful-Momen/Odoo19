# Odoo ORM Controller Cheat Sheet (Student CRUD API)

```text
Controller
    │
    ▼
request.env['student.student'].sudo()
    │
    ├── search()        → GET All
    ├── search_count()  → Total Records
    ├── browse()        → GET One
    ├── exists()        → Check Record
    ├── create()        → POST
    ├── write()         → PUT / PATCH
    └── unlink()        → DELETE
```

---

# 1. search() → Get Multiple Records

## Generic Syntax

```python
# Returns a recordset of matching records

records = self.env['model.name'].search(
    domain=[],
    offset=0,
    limit=None,
    order=None
)
```

## Parameter Meaning

```text
model.name   → Model Name

domain       → Filter Condition

offset       → Skip Records

limit        → Maximum Records

order        → Sort Result
```

## Student Example

### Get All

```python
students = request.env['student.student'].sudo().search([])
```

### Filter

```python
students = request.env['student.student'].sudo().search([
    ('gender','=','male')
])
```

### Limit

```python
students = request.env['student.student'].sudo().search(
    [],
    limit=10
)
```

### Pagination

```python
students = request.env['student.student'].sudo().search(
    [],
    offset=20,
    limit=10
)
```

### Order

```python
students = request.env['student.student'].sudo().search(
    [],
    order='name asc'
)
```

### Controller Use

```text
GET /students
GET /students?page=2
GET /students?gender=male
```

# 3. browse() → Get Record By ID

## Generic Syntax

```python
# Returns record(s) by ID

record = self.env['model.name'].browse(record_id)
```

Multiple IDs

```python
records = self.env['model.name'].browse([1,2,3])
```

## Student Example

```python
student = request.env['student.student'].sudo().browse(student_id)
```

### Exists Check

```python
if not student.exists():

    return {

        "status":404,
        "message":"Student not found"

    }
```

### Controller Use

```text
GET /students/5

PUT /students/5

DELETE /students/5
```

# 4. create() → Create Record

## Generic Syntax

```python
# Creates a new record

record = self.env['model.name'].create({

    'field1':value1,
    'field2':value2,

})
```

## Student Example

```python
student = request.env['student.student'].sudo().create({

    'name':data.get('name'),
    'email':data.get('email'),
    'phone':data.get('phone'),
    'gender':data.get('gender'),

})
```

Return

```python
return {

    "status":201,
    "id":student.id

}
```

### Controller Use

```text
POST /students
```


### Greater Than



### IN


### NOT IN


```python
[

]
```

# ⭐ Master ORM Pattern (Most Used 95%)

```python
# Access Model

request.env['model.name'].sudo()

# GET ALL

.search(
    domain=[],
    offset=0,
    limit=None,
    order=None
)

# COUNT

.search_count(
    domain=[]
)

# GET ONE

.browse(record_id)

.exists()

# CREATE

.create({

    'field':value

})

# UPDATE

.write({

    'field':value

})

# DELETE

.unlink()
```

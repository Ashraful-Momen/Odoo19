# Odoo ORM Controller Cheat Sheet (Student CRUD API)

```text
Controller
    │
    ▼
request.env['student.student'].sudo()
    │
    ├── search()        → GET All
    ├── search_count()  → Total Record
    ├── browse()        → GET One / Update / Delete
    ├── create()        → POST
    ├── write()         → PUT/PATCH
    ├── unlink()        → DELETE
    └── exists()        → Record Exist?
```

---

# 1. Search All Records (GET ALL)

```python
students = request.env['student.student'].sudo().search([])
```

### Filter

```python
students = request.env['student.student'].sudo().search([
    ('gender','=','male')
])
```

### Limit

```python
students = request.env['student.student'].sudo().search(
    [],
    limit=10
)
```

### Pagination

```python
students = request.env['student.student'].sudo().search(
    [],
    offset=20,
    limit=10
)
```

### Order

```python
students = request.env['student.student'].sudo().search(
    [],
    order='name asc'
)
```

# 2. Count Records

```python
total = request.env['student.student'].sudo().search_count([])
```

Filtered

```python
male = request.env['student.student'].sudo().search_count([
    ('gender','=','male')
])
```

---

# 3. Browse (Single Record)

```python
student = request.env['student.student'].sudo().browse(student_id)
```

Exist Check

```python
if not student.exists():
    return {
        "status":404,
        "message":"Student not found"
    }
```

---

# 4. Create Record (POST)

```python
student = request.env['student.student'].sudo().create({

    'name':data.get('name'),
    'email':data.get('email'),
    'phone':data.get('phone'),
    'gender':data.get('gender'),

})
```

Return

```python
return {

    "status":201,
    "id":student.id

}
```

---

# 5. Update Record (PUT)

```python
student.write({

```

Dynamic

```python
student.write(data)
```

Return

```python
return {
```

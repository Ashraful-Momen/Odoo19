# একটি Module কীভাবে Load হয়?

ধরো তোমার `hospital` Module-এর `__manifest__.py`-তে আছে:

```python
{
    "name": "Hospital",
    "depends": ["base", "sale"],
}
```

তখন Odoo এই ক্রমে Module খুঁজবে:

```text
hospital
    │
    ├── base
    │      │
    │      └── odoo/odoo/addons/base
    │
    └── sale
           │
           └── odoo/addons/sale
```

অর্থাৎ `hospital` (custom) Module, `base` (core) এবং `sale` (official addon)-এর উপর নির্ভর করবে।

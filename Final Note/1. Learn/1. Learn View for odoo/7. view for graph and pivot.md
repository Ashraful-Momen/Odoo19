Odoo তে **Graph** আর **Pivot** view data analysis এর জন্য use হয়।

Dashboard, report, chart দেখানোর জন্য এগুলো লাগে।

---

# Graph View কী?

Graph view = bar chart, line chart, pie chart দেখায়।

```text
    Sales
    │    ┌──┐
  50│    │  │    ┌──┐
    │    │  │    │  │
  30│┌──┐│  │    │  │
    ││  ││  │┌──┐│  │
  20││  ││  ││  ││  │
    └─────────────────
      Jan  Feb  Mar
```

---

# Pivot View কী?

Pivot view = Excel pivot table এর মতো।

Row ও Column অনুযায়ী data group করে summary দেখায়।

```text
            │ Jan │ Feb │ Mar │ Total
────────────┼─────┼─────┼─────┼─────
Product A   │ 10  │ 20  │ 15  │ 45
Product B   │ 5   │ 10  │ 8   │ 23
────────────┼─────┼─────┼─────┼─────
Total       │ 15  │ 30  │ 23  │ 68
```

# Complete Graph View Example

```xml
<?xml version="1.0" encoding="UTF-8"?>

<odoo>

    <record id="view_student_graph" model="ir.ui.view">
        <field name="name">student.student.graph</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">
            <graph string="Student Analysis" type="bar">

                <field name="age" type="row"/>
                <field name="id" type="measure"/>

            </graph>
        </field>
    </record>

</odoo>
```

# Graph Line By Line Explanation

---

## 1. Graph Tag

```xml
<graph string="Student Analysis" type="bar">
```

### Attributes

| Attribute | Meaning | Options |
|-----------|---------|---------|
| `string` | Title | any text |
| `type` | Chart type | `bar`, `line`, `pie` |

---

## 2. Row Field

```xml
<field name="age" type="row"/>
```

X-axis এ age দেখাবে।

---

## 3. Measure Field

```xml
<field name="id" type="measure"/>
```

Y-axis এ count দেখাবে।

`id` count = total records।

---

# Graph Types

## Bar Chart (Default)

```xml
<graph string="Students by Age" type="bar">
    <field name="age" type="row"/>
    <field name="id" type="measure"/>
</graph>
```

## Line Chart

```xml
<graph string="Student Trend" type="line">
    <field name="create_date" interval="month" type="row"/>
    <field name="id" type="measure"/>
</graph>
```

## Pie Chart

```xml
<graph string="Student Distribution" type="pie">
    <field name="status" type="row"/>
    <field name="id" type="measure"/>
</graph>
```

# Pivot Line By Line Explanation

---

## 1. Pivot Tag

```xml
<pivot string="Student Pivot">
```

Pivot view শুরু হচ্ছে।

---

## 2. Row Field

```xml
<field name="status" type="row"/>
```

Row হিসেবে status column দেখাবে।

Example:

```text
Active
Inactive
Pending
```

---

## 3. Column Field

```xml
<field name="age" type="col"/>
```

Column হিসেবে age দেখাবে।

---

## 4. Measure Field

```xml
<field name="id" type="measure"/>
```

Cell এ count দেখাবে।

---

# Pivot With Aggregation

```xml
<pivot string="Student Age Analysis">

    <field name="status" type="row"/>
    <field name="age" type="measure" operator="avg"/>

</pivot>
```

`operator="avg"` → average age দেখাবে।

## Common Operators

| Operator | Meaning |
|----------|---------|
| `sum` | Sum of values |
| `avg` | Average |
| `count` | Count |
| `min` | Minimum |
| `max` | Maximum |

```xml
            <field name="amount_total" type="measure"/>
        </pivot>
    </field>

```

# Important Notes

## 1. Date Interval

```xml
<field name="create_date" interval="month" type="row"/>
```

| Interval | Meaning |
|----------|---------|
| `day` | Daily |
| `week` | Weekly |
| `month` | Monthly |
| `quarter` | Quarterly |
| `year` | Yearly |

## 2. Multiple Measures

```xml
<graph string="Analysis" type="bar">
    <field name="status" type="row"/>
    <field name="amount" type="measure"/>
    <field name="quantity" type="measure"/>
</graph>
```

## 3. Graph Color

Odoo automatically different color দেয় different row এর জন্য।

# Interview Questions

## Q: Graph vs Pivot view difference?

Answer:

> Graph = visual chart (bar, line, pie)
> Pivot = table format with row/column grouping

## Q: type="measure" কী করে?

Answer:

> Y-axis (graph) বা cell value (pivot) calculate করে।

## Q: interval="month" কেন দরকার?

Answer:

> Date field কে month-wise group করতে।

# Easy Memory Trick

```text
Graph View:
  <graph type="bar"> = bar chart
  <graph type="line"> = line chart
  <graph type="pie">  = pie chart

Pivot View:
  type="row"    = left side rows
  type="col"    = top columns
  type="measure"= cell values
```

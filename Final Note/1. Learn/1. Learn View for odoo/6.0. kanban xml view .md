# Odoo Kanban View Simple Note

## What is Kanban View in Odoo?

Kanban view is a **card style UI** in Odoo.

Example:

* CRM pipeline cards
* Task board
* Student cards
* Employee cards

Each record shows as a card.

---

# Basic Structure of Kanban

```xml
<kanban>
    <templates>
        <t t-name="card">
            CARD DESIGN
        </t>
    </templates>
</kanban>
```

---

# Step 1: Create Model

## Python Model (`student.py`)

```python
from odoo import models, fields

class Student(models.Model):
    _name = 'student.student'
    _description = 'Student'

    student_id = fields.Char(string="Student ID")

    name = fields.Char(string="Name")

    gender = fields.Selection([
        ('male', 'Male'),
        ('female', 'Female')
    ], string="Gender")

    image = fields.Image(string="Image")
```

# Step 2: Create Menu + Action

## `student_menu.xml`

```xml
<?xml version="1.0" encoding="utf-8"?>

<odoo>

    <!-- Action -->
    <record id="student_action" model="ir.actions.act_window">
        <field name="name">Students</field>

        <field name="res_model">student.student</field>

        <field name="view_mode">kanban,list,form</field>
    </record>

    <!-- Main Menu -->
    <menuitem
        id="student_main_menu"
        name="Student"/>

    <!-- Sub Menu -->
    <menuitem
        id="student_sub_menu"
        name="Students"
        parent="student_main_menu"
        action="student_action"/>

</odoo>
```

# Step 3: Create Kanban View

## `student_kanban_view.xml`

```xml
<?xml version="1.0" encoding="utf-8"?>

<odoo>

    <record id="student_kanban_view" model="ir.ui.view">

        <field name="name">
            student.student.kanban
        </field>

        <field name="model">
            student.student
        </field>

        <field name="arch" type="xml">

            <kanban>

                <!-- Load fields -->
                <field name="student_id"/>
                <field name="name"/>
                <field name="gender"/>
                <field name="image"/>

                <templates>

                    <t t-name="card">

                        <div class="oe_kanban_card p-3">

                            <!-- Image -->
                            <div class="text-center mb-2">

                                <img
                                    t-att-src="kanban_image(
                                        'student.student',
                                        'image',
                                        record.id.raw_value
                                    )"
                                    width="80"
                                    height="80"
                                    class="rounded-circle"/>

                            </div>

                            <!-- Student ID -->
                            <div>

                                <strong>ID:</strong>

                                <field name="student_id"/>

                            </div>

                            <!-- Name -->
                            <div>

                                <strong>Name:</strong>

                                <field name="name"/>

                            </div>

                            <!-- Gender Badge -->
                            <div class="mt-2">

                                <!-- Male -->
                                <span
                                    t-if="record.gender.raw_value == 'male'"
                                    class="badge bg-primary">

                                    Male

                                </span>

                                <!-- Female -->
                                <span
                                    t-if="record.gender.raw_value == 'female'"
                                    class="badge bg-danger">

                                    Female

                                </span>

                            </div>

                        </div>

                    </t>

                </templates>

            </kanban>

        </field>

    </record>

</odoo>
```

# Important Things You Must Know

## 1. `<kanban>`

Start of kanban view.

```xml
<kanban>
</kanban>
```

---

# 2. Load Fields First

Before using fields inside card, load them first.

```xml
<field name="name"/>
```

Without this:

* `record.name.raw_value`
* `<field name="name"/>`

may not work correctly.

---

# 3. `<templates>`

Kanban card design area.

```xml
<templates>
</templates>
```

---

# 4. `t-name="card"`

Defines single card layout.

```xml
<t t-name="card">
```

Every database record uses this template.

---

# 5. `<field name="..."/>`

Shows field value.

Example:

```xml
<field name="name"/>
```

---

# 6. `record.field.raw_value`

Used for conditions.

Example:

```xml
record.gender.raw_value
```

Returns:

* male
* female

---

# 7. `t-if`

Condition rendering.

```xml
t-if="record.gender.raw_value == 'male'"
```

Means:
show this only if gender is male.

---

# 8. `kanban_image()`

Used to show image field.

```xml
kanban_image(
    'student.student',
    'image',
    record.id.raw_value
)
```

Parameters:

| Part                | Meaning           |
| ------------------- | ----------------- |
| student.student     | model name        |
| image               | image field       |
| record.id.raw_value | current record id |

---

# 9. Badge

Bootstrap badge.

```xml
class="badge bg-primary"
```

Common:

* `bg-primary`
* `bg-danger`
* `bg-success`
* `bg-warning`

---

# Final Output

Each student card shows:

* Student image
* Student ID
* Name
* Male/Female badge

Like:

```text
-------------------
|     IMAGE       |
| ID: ST001       |
| Name: Rahim     |
| [Male Badge]    |
-------------------
```

# Very Important Odoo Kanban Concepts

| Concept                  | Usage         |
| ------------------------ | ------------- |
| `<kanban>`               | Start kanban  |
| `<templates>`            | Card design   |
| `t-name="card"`          | Single card   |
| `<field/>`               | Show field    |
| `record.field.raw_value` | Raw data      |
| `t-if`                   | Condition     |
| `kanban_image()`         | Show image    |
| `badge`                  | Colored label |

---

# Odoo 19 Important Note

In Odoo 19:

* Kanban still uses QWeb style templates
* `t-if`
* `t-foreach`
* `record.field.raw_value`

are still very important.

But some OWL-based frontend improvements exist internally.

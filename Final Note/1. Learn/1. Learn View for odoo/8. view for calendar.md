# Complete Calendar View Example

```xml

<odoo>

    <record id="view_student_calendar" model="ir.ui.view">
        <field name="name">student.student.calendar</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">
            <calendar string="Student Calendar"
                      date_start="admission_date"
                      date_stop="graduation_date"
                      color="status"
                      mode="month">

                <field name="name"/>
                <field name="age"/>
                <field name="status"/>

            </calendar>
        </field>
    </record>

</odoo>
```

# Line By Line Bangla Explanation

---

## 1. Calendar Tag

```xml
<calendar string="Student Calendar"
          date_start="admission_date"
          date_stop="graduation_date"
          color="status"
          mode="month">
```

### Attributes Explained

| Attribute | Meaning | Example |
|-----------|---------|---------|
| `string` | Calendar title | any text |
| `date_start` | Event start date field | `admission_date` |
| `date_stop` | Event end date field | `graduation_date` |
| `color` | Group by color | `status` |
| `mode` | Default view mode | `month`, `week`, `day` |

---

## 2. Date Start (Required)

```xml
date_start="admission_date"
```

Calendar এ event কবে start হবে সেটা বলে।

Model এ এই field থাকতে হবে:

```python
admission_date = fields.Date(string="Admission Date")
```

---

## 3. Date Stop (Optional)

```xml
date_stop="graduation_date"
```

Event কবে শেষ হবে।

Multi-day event এর জন্য।

---

## 4. Color

```xml
color="status"
```

Same status এর event same color হবে।

Example:

```text
Active   → Green
Inactive → Red
Pending  → Yellow
```

---

## 5. Mode

```xml
mode="month"
```

Default calendar mode:

| Mode | View |
|------|------|
| `month` | Monthly grid |
| `week` | Weekly grid |
| `day` | Daily list |

---

## 6. Fields Inside Calendar

```xml
<field name="name"/>
<field name="age"/>
<field name="status"/>
```

Event card এ এই fields দেখাবে।

# Simple Calendar (Single Day Event)

```xml
<calendar string="Student Admission"
          date_start="admission_date"
          color="status"
          mode="month">

    <field name="name"/>

</calendar>
```

একদিনের event এর জন্য শুধু `date_start` enough।

---

# Calendar With All Day

```xml
<calendar string="Holidays"
          date_start="holiday_date"
          all_day="1"
          mode="month">

    <field name="name"/>

</calendar>
```

`all_day="1"` → পুরো দিন event হিসেবে দেখাবে।

---

# Quick Create In Calendar

```xml
<calendar string="Events"
          date_start="start_date"
          quick_add="1"
          mode="month">

    <field name="name"/>

</calendar>
```

`quick_add="1"` → calendar এ click করলে quick create popup দেখাবে।

# Model Requirement

Calendar view এর জন্য model এ **minimum 1 date field** লাগে:

```python
from odoo import models, fields

class Student(models.Model):
    _name = 'student.student'

    name = fields.Char(string="Name", required=True)
    admission_date = fields.Date(string="Admission Date")
    graduation_date = fields.Date(string="Graduation Date")
    status = fields.Selection([
        ('active', 'Active'),
        ('inactive', 'Inactive'),
    ], string="Status")
```

# Real Industry Example

```xml
<record id="view_meeting_calendar" model="ir.ui.view">
    <field name="name">meeting.calendar</field>
    <field name="model">calendar.event</field>
    <field name="arch" type="xml">
        <calendar string="Meetings"
                  date_start="start"
                  date_stop="stop"
                  date_delay="duration"
                  color="user_id"
                  mode="week">

            <field name="name"/>
            <field name="partner_id"/>
            <field name="user_id"/>

        </calendar>
    </field>
</record>
```

## Extra Attributes Used

| Attribute | Meaning |
|-----------|---------|
| `date_delay` | Event duration in hours |
| `user_id` | Assignee (color by user) |

# Interview Questions

## Q: Calendar view কখন use করো?

Answer:

> Date-based data দেখানোর জন্য — meeting, appointment, deadline, schedule।

## Q: date_start vs date_stop difference?

Answer:

> `date_start` = event start (required)
> `date_stop` = event end (optional, for multi-day events)

## Q: color attribute কী করে?

Answer:

> Same value এর event same color দেয়, যা visually group করা সহজ করে।

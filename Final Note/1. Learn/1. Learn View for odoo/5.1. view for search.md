Odoo তে **Search View** খুব important কারণ user data filter করতে পারে।

Search view ছাড়া user সব record একসাথে দেখতে পাবে, যা inconvenient।

---

# Search View কী?

Search view = UI তে **filter, group by, search field** দেখায়।

```text
┌─────────────────────────────────────────┐
│ [Search...]  🔍 Filter  Group By  ❌   │
└─────────────────────────────────────────┘
```

---

# Complete Search View Example

```xml
<?xml version="1.0" encoding="UTF-8"?>

<odoo>

    <record id="view_student_search" model="ir.ui.view">
        <field name="name">student.student.search</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">
            <search string="Search Students">

                <!-- Search Fields -->
                <field name="name"/>
                <field name="email"/>

                <!-- Filters -->
                <filter
                    string="Adult"
                    name="filter_adult"
                    domain="[('age', '>=', 18)]"
                />

                <filter
                    string="Minor"
                    name="filter_minor"
                    domain="[('age', '<', 18)]"
                />

                <!-- Group By -->
                <group expand="0" string="Group By">
                    <filter
                        string="Age"
                        name="group_by_age"
                        context="{'group_by': 'age'}"
                    />
                </group>

            </search>
        </field>
    </record>

</odoo>
```

# Line By Line Bangla Explanation

---

## 1. Record Create

```xml
<record id="view_student_search" model="ir.ui.view">
```

Search view create হচ্ছে।

`model="ir.ui.view"` মানে এটা view record।

---

## 2. View Model

```xml
<field name="model">student.student</field>
```

কোন model এর জন্য search view কাজ করবে।

---

## 3. Search Tag

```xml
<search string="Search Students">
```

Search view শুরু হচ্ছে।

`string` = placeholder text।

---

## 4. Search Field

```xml
<field name="name"/>
```

User `name` দিয়ে search করতে পারবে।

Example:

```text
Search: [John] → John name যে সব student তাদে দেখাবে
```

---

## 5. Filter

```xml
<filter
    string="Adult"
    name="filter_adult"
    domain="[('age', '>=', 18)]"
/>
```

### Meaning

| Attribute | Meaning |
|-----------|---------|
| `string` | UI তে দেখা নাম |
| `name` | unique identifier |
| `domain` | filter condition |

এই filter click করলে শুধু adult student (age >= 18) দেখাবে।

---

## 6. Group By

```xml
<group expand="0" string="Group By">
    <filter
        string="Age"
        name="group_by_age"
        context="{'group_by': 'age'}"
    />
</group>
```

### Meaning

`group expand="0"` → group by section collapsed থাকবে।

`context="{'group_by': 'age'}"` → age অনুযায়ী group করবে।

# Domain Explained

Domain = Odoo এর filter condition syntax।

```python
# SQL equivalent
WHERE age >= 18

# Odoo domain
domain="[('age', '>=', 18)]"
```

## Common Operators

| Operator | Meaning |
|----------|---------|
| `=` | equal |
| `!=` | not equal |
| `>` | greater than |
| `<` | less than |
| `>=` | greater or equal |
| `<=` | less or equal |
| `in` | in list |
| `like` | contains |
| `ilike` | contains (case insensitive) |

---

# Multiple Domain (AND)

```xml
<filter
    string="Adult with Email"
    name="filter_adult_email"
    domain="[('age', '>=', 18), ('email', '!=', False)]"
/>
```

দুইটা condition একসাথে:

* age >= 18
* email not empty

---

# OR Domain

```xml
<filter
    string="Young or Old"
    name="filter_young_old"
    domain="['|', ('age', '<', 13), ('age', '>=', 60)]"
/>
```

`'` `|` `'` = OR operator।

Default behavior = AND।

# Real Industry Example

```xml
<search string="Search Students">

    <field name="name"/>
    <field name="email"/>
    <field name="age"/>

    <separator/>

    <filter
        string="Active"
        name="filter_active"
        domain="[('active', '=', True)]"
    />

    <filter
        string="Archived"
        name="filter_archived"
        domain="[('active', '=', False)]"
    />

    <group expand="0" string="Group By">
        <filter string="Age" name="group_age" context="{'group_by': 'age'}"/>
        <filter string="Status" name="group_status" context="{'group_by': 'status'}"/>
    </group>

</search>
```

# Important Tips

## 1. Search View আলাদা File এ রাখো

```text
views/
    student_views.xml      # list + form
    student_search.xml     # search view
```

## 2. Manifest এ Add করো

```python
'data': [
    'views/student_views.xml',
    'views/student_search.xml',
],
```

## 3. Action এ Search View Link করো

```xml
<record id="action_student" model="ir.actions.act_window">
    <field name="name">Students</field>
    <field name="res_model">student.student</field>
    <field name="view_mode">list,form</field>
    <field name="search_view_id" ref="view_student_search"/>
</record>
```

`search_view_id` = কোন search view use হবে।

# Interview Questions

## Q: Domain কী?

Answer:

> Odoo এর filter condition যা record filter করতে ব্যবহার হয়।

## Q: context="{'group_by': 'field'}" কী করে?

Answer:

> Record কে specific field অনুযায়ী group করে দেখায়।

## Q: separator tag কী?

Answer:

> Filter এর মাঝে divider line দেয়।

# Easy Memory Trick

<group>     = group by section

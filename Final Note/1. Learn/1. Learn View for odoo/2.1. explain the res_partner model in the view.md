eta bujini => partner_id সাধারণত Many2one relation।
এটা res.partner model এর সাথে linked।

=========================================================


ধরো Odoo তে:

* Customer information আলাদা table/model এ থাকে
* Sales Order আলাদা table/model এ থাকে

এখন Sales Order এর ভিতরে customer এর পুরো data save না করে শুধু customer এর reference save করা হয়।

এটাই `Many2one`.

---

# সহজভাবে Concept

একজন customer অনেকগুলো order করতে পারে।

মানে:

```text
1 Customer → Many Sales Orders
```

কিন্তু:

```text
1 Sales Order → Only 1 Customer
```

এই relation টাই Odoo তে `Many2one`.

---

# Example

## Customer Table (`res.partner`)

| id | name  |
| -- | ----- |
| 1  | Rahim |
| 2  | Karim |

---

## Sales Order Table (`sale.order`)

| id  | partner_id | amount |
| --- | ---------- | ------ |
| 101 | 1          | 500    |
| 102 | 1          | 700    |
| 103 | 2          | 300    |

---

এখানে:

```text
partner_id = 1
```

মানে:

```text
এই order Rahim এর
```

কারণ:

* `1` হলো `res.partner` table এর id।

---

# এখন Python Model দেখো

```python
partner_id = fields.Many2one("res.partner")
```

### এর মানে:

| অংশ               | Meaning                    |
| ----------------- | -------------------------- |
| `fields.Many2one` | relation field             |
| `"res.partner"`   | কোন model এর সাথে relation |

---

# Real Meaning

```python
partner_id = fields.Many2one("res.partner")
```

মানে:

> “sale.order model এর ভিতরে এমন একটি field তৈরি করো যা res.partner model এর একটি record কে reference করবে।”

---

# Database এ আসলে কি save হয়?

অনেকে ভাবে পুরো customer object save হয়।

না।

শুধু ID save হয়।

Example:

```text
partner_id = 5
```

শুধু integer foreign key save হয়।

---

# কেন নাম `partner_id`?

Odoo naming convention:

```text
relation_field + _id
```

কারণ এটা অন্য model এর ID store করে।

---

# XML View এ কি হয়?

```xml
<field name="partner_id"/>
```

Odoo automatically বুঝে:

* এটা Many2one field
* তাই dropdown/search box render করবে

# Many2one নাম কেন?

কারণ:

```text
Many Orders → One Customer
```

মানে:

* অনেক sale.order
* একটি res.partner

তাই নাম:

```text
Many2one
```

---

# Odoo Relation Types

| Type      | Meaning     |
| --------- | ----------- |
| Many2one  | Many → One  |
| One2many  | One → Many  |
| Many2many | Many ↔ Many |

---

# Example Together

## Many2one

```python
partner_id = fields.Many2one("res.partner")
```

একটি order → একটি customer

---

## One2many

```python
order_ids = fields.One2many("sale.order", "partner_id")
```

একটি customer → অনেক order

---

# Relationship দুইদিক থেকে

```text
Customer → অনেক Orders
Order → একটি Customer
```

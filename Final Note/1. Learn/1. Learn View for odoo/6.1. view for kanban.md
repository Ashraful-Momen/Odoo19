Odoo তে **Kanban View** card-based UI তৈরি করে।

Dashboard, task board, CRM pipeline এ kanban খুব popular।

---

# Kanban View কী?

Kanban view = card layout যেখানে প্রতিটা record একটা card হিসেবে দেখায়।

```text
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│ 📋 John     │  │ 📋 Jane     │  │ 📋 Mike     │
│ Age: 20     │  │ Age: 22     │  │ Age: 18     │
│ john@test   │  │ jane@test   │  │ mike@test   │
└─────────────┘  └─────────────┘  └─────────────┘
```

# Complete Kanban View Example

```xml
<?xml version="1.0" encoding="UTF-8"?>

<odoo>

    <record id="view_student_kanban" model="ir.ui.view">
        <field name="name">student.student.kanban</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">
            <kanban string="Students">

                <!-- Card Template -->
                <templates>
                    <t t-name="kanban-box">

                        <div class="oe_kanban_global_click">

                            <!-- Top Section -->
                            <div class="o_kanban_record_top">
                                <div class="o_kanban_record_headings">
                                    <strong class="o_kanban_record_title">
                                        <field name="name"/>
                                    </strong>
                                </div>
                            </div>

                            <!-- Body Section -->
                            <div class="o_kanban_record_body">
                                <div>
                                    Age: <field name="age"/>
                                </div>
                                <div>
                                    Email: <field name="email"/>
                                </div>
                            </div>

                            <!-- Bottom Section -->
                            <div class="o_kanban_record_bottom">
                                <span class="oe_kanban_list_many2many">
                                    <field name="id"/>
                                </span>
                            </div>

                        </div>

                    </t>
                </templates>

            </kanban>
        </field>
    </record>

</odoo>
```

# Line By Line Bangla Explanation

---

## 1. Kanban Record

```xml
<record id="view_student_kanban" model="ir.ui.view">
```

Kanban view create হচ্ছে।

---

## 2. Kanban Tag

```xml
<kanban string="Students">
```

Kanban view শুরু হচ্ছে।

---

## 3. Templates

```xml
<templates>
```

Card design template।

প্রতিটা record এই template অনুযায়ী render হবে।

---

## 4. Template Name

```xml
<t t-name="kanban-box">
```

Odoo default kanban template name = `kanban-box`।

এটা change করো না।

---

## 5. Clickable Card

```xml
<div class="oe_kanban_global_click">
```

পুরো card clickable হবে।

Click করলে form view open হবে।

---

## 6. Record Title

```xml
<strong class="o_kanban_record_title">
    <field name="name"/>
</strong>
```

Card এর main title।

Bold হিসেবে দেখাবে।

---

## 7. Record Body

```xml
<div class="o_kanban_record_body">
    <div>
        Age: <field name="age"/>
    </div>
</div>
```

Card এর body content।

Label + field value দেখাবে।

---

## 8. Field Tag Inside Kanban

```xml
<field name="email"/>
```

Model field এর value দেখাবে।

# Kanban With Color and Status

```xml
<kanban string="Students">

    <!-- Fields needed in kanban -->
    <field name="name"/>
    <field name="age"/>
    <field name="email"/>
    <field name="status"/>

    <templates>
        <t t-name="kanban-box">

            <div t-attf-class="oe_kanban_global_click">

                <!-- Color Badge -->
                <div class="o_kanban_tags">
                    <span class="badge bg-primary">
                        <field name="status"/>
                    </span>
                </div>

                <!-- Title -->
                <div class="o_kanban_record_top">
                    <strong>
                        <field name="name"/>
                    </strong>
                </div>

                <!-- Details -->
                <div class="o_kanban_record_body">
                    <div>Age: <field name="age"/></div>
                    <div>Email: <field name="email"/></div>
                </div>

            </div>

        </t>
    </templates>

</kanban>
```

# Action With Kanban

```xml
<record id="action_student" model="ir.actions.act_window">
    <field name="name">Students</field>
    <field name="res_model">student.student</field>
    <field name="view_mode">kanban,list,form</field>
</record>
```

`view_mode="kanban,list,form"` → প্রথমে kanban দেখাবে।

---

# Important CSS Classes

| Class | Meaning |
|-------|---------|
| `oe_kanban_global_click` | পুরো card clickable |
| `o_kanban_record_top` | Top section |
| `o_kanban_record_body` | Body section |
| `o_kanban_record_bottom` | Bottom section |
| `o_kanban_record_title` | Title styling |
| `oe_kanban_color_1` to `_11` | Color coding |

---

# Color Coded Kanban

```xml
<kanban string="Students">

    <field name="name"/>
    <field name="priority"/>

    <templates>
        <t t-name="kanban-box">

            <div t-attf-class="oe_kanban_color_#{record.priority.raw_value}
                                oe_kanban_global_click">

                <field name="name"/>

            </div>

        </t>
    </templates>

</kanban>
```

`priority` field অনুযায়ী card এর color change হবে।

---

# Quick Create

```xml
<kanban string="Students" quick_create="true">
```

`quick_create="true"` → card এ direct create button দেখাবে।

# Interview Questions

## Q: Kanban view কী?

Answer:

> Card-based UI যেখানে প্রতিটা record একটা card হিসেবে দেখায়।

## Q: `t-name="kanban-box"` কেন দরকার?

Answer:

> Odoo default kanban template identifier। এটা ছাড়া card render হয় না।

## Q: Kanban এ click কিভাবে কাজ করে?

Answer:

> `oe_kanban_global_click` class use করলে card click করলে form view open হয়।

# Easy Memory Trick

```text
<kanban>     = card layout start
<templates>  = card design container
<t t-name>   = template definition
<div class>  = HTML structure of card
<field>      = dynamic data from model
```

Odoo form view এ অনেক field থাকলে **Notebook** use করা হয় tab organization এর জন্য।

Long form গুলোকে section-wise divide করতে notebook perfect solution।

---

# Notebook কী?

Notebook = Tab system যেখানে প্রতিটা tab = একটা page।

```text
┌──────────────────────────────────────┐
│ [General] [Education] [Documents]   │  ← Tabs
├──────────────────────────────────────┤
│                                      │
│  Name:    [John Doe          ]      │
│  Email:   [john@test.com     ]      │
│  Phone:   [+1234567890       ]      │
│                                      │
└──────────────────────────────────────┘
```

# Complete Notebook Example

```xml
<?xml version="1.0" encoding="UTF-8"?>

<odoo>

    <record id="view_student_form_with_notebook" model="ir.ui.view">
        <field name="name">student.student.form.notebook</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">
            <form string="Student Form">

                <sheet>

                    <!-- Top Section (Always Visible) -->
                    <group>
                        <field name="name"/>
                        <field name="email"/>
                    </group>

                    <!-- Notebook / Tabs -->
                    <notebook>

                        <!-- Tab 1: Personal Info -->
                        <page string="Personal Info">
                            <group>
                                <field name="age"/>
                                <field name="phone"/>
                                <field name="address"/>
                            </group>
                        </page>

                        <!-- Tab 2: Education -->
                        <page string="Education">
                            <group>
                                <field name="school_name"/>
                                <field name="grade"/>
                                <field name="gpa"/>
                            </group>
                        </page>

                        <!-- Tab 3: Documents -->
                        <page string="Documents">
                            <group>
                                <field name="id_card"/>
                                <field name="certificate"/>
                            </group>
                        </page>

                    </notebook>

                </sheet>

            </form>
        </field>
    </record>

</odoo>
```

# Line By Line Bangla Explanation

---

## 1. Notebook Tag

```xml
<notebook>
```

Tab container শুরু হচ্ছে।

সব page/tab এই tag এর ভিতরে থাকে।

---

## 2. Page Tag

```xml
<page string="Personal Info">
```

একটা tab/page তৈরি করছে।

`string` = tab title যা UI তে দেখা যাবে।

---

## 3. Page Content

```xml
<page string="Personal Info">
    <group>
        <field name="age"/>
        <field name="phone"/>
    </group>
</page>
```

প্রতিটা page এর ভিতরে normal form content থাকে:

* `<group>`
* `<field>`
* `<div>`
* যেকোনো form element

---

## 4. Page Close

```xml
</page>
```

Page/tab end হচ্ছে।

---

## 5. Notebook Close

```xml
</notebook>
```

পুরো tab container end হচ্ছে।

# Chatter Section

```xml
<div class="oe_chatter">
    <field name="message_follower_ids"/>
    <field name="activity_ids"/>
    <field name="message_ids"/>
</div>
```

Chatter = Message + Activity + Followers section।

```text
┌─────────────────────────────────────────┐
│ Messages & Activities                   │
├─────────────────────────────────────────┤
│ Followers: [Admin] [Manager]            │
│                                         │
│ [Schedule Activity]                     │
│                                         │
│ John: Student updated                   │
│ Jane: Please review documents           │
│                                         │
│ [Send Message] [Log Note]               │
└─────────────────────────────────────────┘
```

## Model Requirement for Chatter

Python model এ inherit করতে হবে:

```python
from odoo import models, fields

class Student(models.Model):
    _name = 'student.student'
    _inherit = ['mail.thread', 'mail.activity.mixin']

    name = fields.Char(string="Name")
    # ... other fields
```

`_inherit = ['mail.thread', 'mail.activity.mixin']` → chatter enable করে।

# Important Tips

## 1. Notebook Position

Notebook normally form এর middle বা bottom এ রাখো:

```text
Form Top    → Always visible info
    ↓
Notebook    → Tab sections
    ↓
Chatter     → Messages (optional)
```

## 2. Page Name Clear রাখো

```xml
<!-- Good -->
<page string="Personal Information">

<!-- Bad -->
<page string="Tab 1">
```

## 3. Group Inside Page

প্রতিটা page এর field গুলো `<group>` দিয়ে organize করো।

## 4. Active Tab

Default active tab = first page।

# Interview Questions

## Q: Notebook কী?

Answer:

> Form view এ tab system যা content কে section-wise organize করে।

## Q: Page vs Notebook difference?

Answer:

> `<notebook>` = tab container
> `<page>` = individual tab

## Q: Chatter কেন use করা হয়?

Answer:

> Message, activity, follower tracking এর জন্য। Model এ `mail.thread` inherit করতে হয়।

## Q: Tab এর মধ্যে দুই column কিভাবে?

Answer:

> `<group>` এর ভিতরে দুইটা `<group>` use করলে auto two-column হয়।

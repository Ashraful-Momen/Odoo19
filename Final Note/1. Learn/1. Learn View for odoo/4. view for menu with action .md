ধরো তুমি Odoo তে একটা simple module বানাচ্ছো যেখানে:

* একটা menu থাকবে
* menu click করলে list/form view open হবে
* model হবে `student.student`

এখন আমি তোমাকে complete flow দেখাচ্ছি।

---

# Odoo View/XML Basic Flow

```text
Menu
   ↓
Action
   ↓
View (list/form)
   ↓
Model Data
```

---

# Complete Simple XML Example

```xml
<?xml version="1.0" encoding="UTF-8"?>

<odoo>

    <!-- =========================
            LIST VIEW
    ========================== -->

    <record id="view_student_list" model="ir.ui.view">
        <field name="name">student.student.list</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">
            <list string="Students">

                <field name="name"/>
                <field name="email"/>
                <field name="age"/>

            </list>
        </field>
    </record>


    <!-- =========================
            FORM VIEW
    ========================== -->

    <record id="view_student_form" model="ir.ui.view">
        <field name="name">student.student.form</field>
        <field name="model">student.student</field>

        <field name="arch" type="xml">

            <form string="Student Form">

                <sheet>

                    <group>

                        <field name="name"/>
                        <field name="email"/>
                        <field name="age"/>

                    </group>

                </sheet>

            </form>

        </field>
    </record>


    <!-- =========================
            ACTION
    ========================== -->

    <record id="action_student" model="ir.actions.act_window">

        <field name="name">Students</field>

        <field name="res_model">student.student</field>

        <field name="view_mode">list,form</field>

    </record>


    <!-- =========================
            MAIN MENU
    ========================== -->

    <menuitem
        id="menu_student_root"
        name="Student Management"
        sequence="1"
    />


    <!-- =========================
            SUB MENU
    ========================== -->

    <menuitem
        id="menu_student"
        name="Students"
        parent="menu_student_root"
        action="action_student"
        sequence="1"
    />

</odoo>
```

---

# এখন Line By Line Bangla Explanation

---

# 1. XML Declaration

```xml
<?xml version="1.0" encoding="UTF-8"?>
```

এটা XML file start declaration।

বলছে:

* XML version 1.0
* UTF-8 encoding use হবে

প্রায় সব XML file এ এটা থাকে।

---

# 2. Root Tag

```xml
<odoo>
```

সব Odoo XML এই tag এর ভিতরে থাকে।

এটা container।

শেষে close হবে:

```xml
</odoo>
```

---

# LIST VIEW PART

---

# 3. Record Create

```xml
<record id="view_student_list" model="ir.ui.view">
```

এখানে নতুন একটা database record create হচ্ছে।

## Meaning

| Part               | Meaning          |
| ------------------ | ---------------- |
| record             | DB record create |
| id                 | unique xml id    |
| model="ir.ui.view" | এটা একটা view    |

---

# 4. View Name

```xml
<field name="name">student.student.list</field>
```

Internal technical name।

User normally এটা দেখে না।

---

# 5. Model

```xml
<field name="model">student.student</field>
```

এই view কোন model এর জন্য কাজ করবে।

মানে:

```python
class Student(models.Model):
    _name = "student.student"
```

এই model এর data দেখাবে।

---

# 6. Architecture Field

```xml
<field name="arch" type="xml">
```

এখানে actual UI লেখা হয়।

সব form/list UI এইখানে থাকে।

---

# 7. List View

```xml
<list string="Students">
```

Odoo 19 এ `tree` এর বদলে `list`

এটা table view তৈরি করে।

---

# 8. Fields

```xml
<field name="name"/>
```

মানে model এর `name` field দেখাও।

যদি model এ থাকে:

```python
name = fields.Char()
```

তাহলে এটা render হবে।

---

# Final Output

```text
+----------------------+
| Name | Email | Age   |
+----------------------+
| John | ...   | 20    |
+----------------------+
```

# ACTION PART

---

# 14. Action Record

```xml
<record id="action_student"
        model="ir.actions.act_window">
```

এটা সবচেয়ে important।

Action define করে:

* কোন model open হবে
* কোন view open হবে

---

# 15. Action Name

```xml
<field name="name">Students</field>
```

Menu click করলে title হিসেবে দেখাবে।

---

# 16. res_model

```xml
<field name="res_model">student.student</field>
```

কোন model open হবে।

---

# 17. view_mode

```xml
<field name="view_mode">list,form</field>
```

মানে:

* প্রথমে list open হবে
* record click করলে form open হবে

---

# 19. Sub Menu

```xml

    id="menu_student"

# 20. Parent

```

# Common Interview Question

## Difference Between Menu And Action?

| Menu           | Action                   |
| -------------- | ------------------------ |
| clickable item | what happens after click |
| UI navigation  | open model/view          |
| menuitem       | ir.actions.act_window    |

---

# Most Important Things To Remember

## Menu only UI item

```xml
<menuitem />
```

## Action opens screen

```xml
<record model="ir.actions.act_window">
```

## View defines design

```xml
<record model="ir.ui.view">
```

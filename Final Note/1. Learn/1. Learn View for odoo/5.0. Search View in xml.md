# Simple Search View Tutorial (Odoo 19)

## Step 1: Add `student_id` to Model

`models/models.py`:

```python
from odoo import models, fields

class Student(models.Model):
    _name = 'student_management.student_management'
    _description = 'student information'

    student_id = fields.Char(string="Student ID", required=True)
    name = fields.Char(string="Name", required=True)
    age = fields.Integer(string="Age")
    email = fields.Char(string="Email")
```

**Concept:** `student_id` field add করলে প্রতিটা student এর জন্য unique ID থাকবে।

---

## Step 2: Update List View + Form View

`student_id` field list এবং form view তে add করতে হবে যাতে দেখা যায়:

```xml
<list>
    <field name="student_id"/>
    <field name="name"/>
    <field name="age"/>
    <field name="email"/>
</list>
```

```xml
<form>
    <sheet>
        <group>
            <field name="student_id"/>
            <field name="name"/>
            <field name="age"/>
            <field name="email"/>
        </group>
    </sheet>
</form>
```

---

## Step 3: Create Search View

```xml
<record id="view_student_search" model="ir.ui.view">
    <field name="name">student.search</field>
    <field name="model">student_management.student_management</field>
    <field name="arch" type="xml">
        <search>
            <field name="student_id"/>
            <field name="name"/>
        </search>
    </field>
</record>
```

**Concept:**

| Tag | Meaning |
|-----|---------|
| `<search>` | Search view start |
| `<field name="student_id"/>` | Student ID দিয়ে search করা যাবে |
| `<field name="name"/>` | Name দিয়ে search করা যাবে |

## Complete views/student_view.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<odoo>

    <!-- List View -->
    <record id="view_student_tree" model="ir.ui.view">
        <field name="name">student.tree</field>
        <field name="model">student_management.student_management</field>
        <field name="arch" type="xml">
            <list>
                <field name="student_id"/>
                <field name="name"/>
                <field name="age"/>
                <field name="email"/>
            </list>
        </field>
    </record>

    <!-- Form View -->
    <record id="view_student_form" model="ir.ui.view">
        <field name="name">student.form</field>
        <field name="model">student_management.student_management</field>
        <field name="arch" type="xml">
            <form>
                <sheet>
                    <group>
                        <field name="student_id"/>
                        <field name="name"/>
                        <field name="age"/>
                        <field name="email"/>
                    </group>
                </sheet>
            </form>
        </field>
    </record>

    <!-- Search View -->
    <record id="view_student_search" model="ir.ui.view">
        <field name="name">student.search</field>
        <field name="model">student_management.student_management</field>
        <field name="arch" type="xml">
            <search>
                <field name="student_id"/>
                <field name="name"/>
            </search>
        </field>
    </record>

    <!-- Action -->
    <record id="action_student" model="ir.actions.act_window">
        <field name="name">Students</field>
        <field name="res_model">student_management.student_management</field>
        <field name="view_mode">list,form</field>
    </record>

    <!-- Menu -->
    <menuitem id="menu_student_root" name="Student"/>
    <menuitem id="menu_student" name="Students" parent="menu_student_root" action="action_student"/>

</odoo>
```

## UI Result

```text
┌─────────────────────────────────────────────┐
│ Search...                                   │
├─────────────────────────────────────────────┤
│ Student ID │ Name      │ Age │ Email        │
├─────────────────────────────────────────────┤
│ STD001     │ John      │ 20  │ john@test    │
│ STD002     │ Jane      │ 22  │ jane@test    │
│ STD003     │ Mike      │ 18  │ mike@test    │
└─────────────────────────────────────────────┘
```

**Search behavior:**
- `STD001` লিখে search → STD001 দেখাবে
- `John` লিখে search → John দেখাবে
- `ST` লিখে search → STD001, STD002, STD003 (partial match)

## Important Tips

1. **Search view আলাদা file এ রাখা যায়:**
   ```text
   views/
       student_view.xml    # list + form + search (combined)
       student_search.xml  # search only (separate)
   ```

2. **Action এ search_view_id link করো (optional):**
   ```xml
   <record id="action_student" model="ir.actions.act_window">
       <field name="name">Students</field>
       <field name="res_model">student_management.student_management</field>
       <field name="view_mode">list,form</field>
       <field name="search_view_id" ref="view_student_search"/>
   </record>
   ```

3. **Multiple search fields add করা যায়:**
   ```xml
   <search>
       <field name="student_id"/>
       <field name="name"/>
       <field name="email"/>
   </search>
   ```

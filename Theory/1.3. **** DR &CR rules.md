হ্যাঁ। Accounting-এর সবচেয়ে কঠিন অংশই হলো **Debit (Dr) এবং Credit (Cr)**। কিন্তু একবার **Golden Rules** বুঝে গেলে পুরো Odoo Accounting সহজ হয়ে যাবে।

---

# প্রথমে Dr/Cr মানে কী?

**Debit (Dr)** = কোনো Account **বাড়ছে** অথবা **Value গ্রহণ করছে**

**Credit (Cr)** = কোনো Account **কমছে** অথবা **Value দিচ্ছে**

কিন্তু সব Account-এর জন্য এই নিয়ম একই না।

এজন্য Account-কে ৫ ভাগে ভাগ করা হয়।

```
Assets
Liabilities
Equity
Income
Expense
```

---

# Accounting Formula

```
Assets = Liabilities + Equity
```

আর Profit হিসাব হয়

```
Profit = Income - Expense
```

---

# সবচেয়ে গুরুত্বপূর্ণ Shortcut (এটাই মুখস্থ করুন)

```
Account Type         Increase        Decrease

Assets                 Dr              Cr

Expenses               Dr              Cr

Liabilities            Cr              Dr

Equity                 Cr              Dr

Income                 Cr              Dr
```

এক লাইনে মনে রাখুন:

```
Dr = Asset + Expense

Cr = Liability + Equity + Income
```

অথবা আরও ছোট করে:

```
DEA LER

D = Debit
E = Expense
A = Asset

L = Liability
E = Equity
R = Revenue(Income)
```

এটাই Accounting-এর সবচেয়ে জনপ্রিয় mnemonic।

# Accounting Cheat Sheet (এটি মুখস্থ করুন)

```
                Increase     Decrease

Asset              Dr           Cr

Expense            Dr           Cr

Liability          Cr           Dr

Equity             Cr           Dr

Income             Cr           Dr
```

অর্থাৎ—

```
        D E A D
       /  |  \
   Debit |  Asset
          |
      Expense

        C L E R
       /  | |  \
 Credit Liability Equity Revenue
```

এটাই পুরো Double Entry Accounting-এর ভিত্তি। Odoo 19-এ যখনই কোনো Invoice, Bill, Payment, Journal Entry বা Bank Reconciliation হবে, শেষ পর্যন্ত এই নিয়মগুলিই প্রয়োগ হবে। একবার এই pattern আয়ত্ত করতে পারলে Odoo Accounting-এর ৯০% Journal Entry নিজেই বুঝতে পারবেন।
